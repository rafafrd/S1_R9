# 🧠 Exercícios Básicos com Arrays em JavaScript

Este repositório reúne dois exercícios introdutórios que exploram o uso de arrays e estruturas de repetição na linguagem JavaScript.

---

## 📌 Exercício 1 – Preenchimento de Array

Neste exercício, o objetivo é criar um array com 10 posições e preenchê-lo com valores numéricos de 0 a 9 utilizando um laço de repetição `for`. Ao final, o conteúdo do array é exibido no console. Esse exemplo ajuda a entender como acessar e atribuir valores em posições específicas de um array.

---

## 📌 Exercício 2 – Cálculo da Média de um Aluno

O segundo exercício simula um sistema simples de cálculo de média escolar. O programa:

- Solicita o nome do aluno.
- Coleta quatro notas, uma para cada bimestre.
- Valida se a nota digitada é um número.
- Soma todas as notas e calcula a média.
- Exibe o resultado final com a média e as notas fornecidas.

Esse exercício reforça o uso de arrays, loops, validações com `isNaN` e a interação com o usuário via `prompt()` e `alert()` (ambiente de navegador).

---

## 🧩 Conceitos praticados

- Declaração e manipulação de arrays
- Laços de repetição (`for`)
- Interação com o usuário via `prompt` e `alert`
- Validação de dados com `isNaN`
- Operações matemáticas simples (soma e média)
